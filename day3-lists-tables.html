Nested lists

You can nest ordered and unordered lists to create hierarchical lists. 

<ul>
<li>Fruits
<ul>
<li>Apple</li>
<li>Banana</li>
</ul>
</li>
<li>Vegetables
<ol>
<li>Carrot</li>
<li>Spinach</li>
</ol>
</li>
</ul>

Tables

Tables display tabular data using <table>, <tr>, <th>, and <td>. 

<table border="1">
<tr>
<th>Name</th>
<th>Role</th>
</tr>
<tr>
<td>Alice</td>
<td>Developer</td>
</tr>
</table>

Complex example: list inside table cell

<table border="1">
<tr><th>Section</th><th>Items</th></tr>
<tr>
<td>Fruits & Veg</td>
<td>
<ul>
<li>Apple</li>
<li>Carrot</li>
</ul>
</td>
</tr>
</table>
