🚀 My 6-Day HTML Learning Journey

Welcome! This project documents my 6-day journey of learning HTML.
Click on a day below to explore what I learned 👇



📚 Daily Lessons

  • Day 01 — Introduction to Web Development
    Learn the basics of web development

  • Day 02 — What is HTML?
    Understand HTML structure and tags

  • Day 03 — HTML Nested Lists & Tables
    Create organized content with lists and tables

  • Day 04 — File Path in HTML
    Master linking between files and resources

  • Day 05 — HTML Forms: Beginner → Advanced
    Build interactive forms for user input

  • Day 06 — Deploy Your First Project
    Publish your website to the world


📖 How to Use This Project

  1. 💾 Save each file block into its named .html file.

  2. 🌐 Open index.html in your browser to navigate lessons.

  3. 📝 Explore pure HTML examples (no CSS or JavaScript).
