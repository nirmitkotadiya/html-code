What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses tags to describe elements such as headings, paragraphs, links, images, lists, and forms.

Breaking It Down

  • HyperText: Links connect web pages together, forming the "web".
  • Markup Language: Tags mark up plain text to give it structure and meaning.
