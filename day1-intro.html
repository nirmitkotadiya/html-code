💡 What is Web Development?

Web development is building websites and web applications that run on browsers. It commonly involves three areas:

  • HTML — structure and content
  • CSS — presentation (not used in this project)
  • JavaScript — behavior (not used here)

⚙️ How Web Pages Work (Simple)

  1. Your browser requests an .html file from a server or local disk.
  2. The browser parses the HTML and displays content.
  3. Links connect pages to create a multi-page site.

🌍 HOW Does Internet Work

🔑 Principle 1: The Need to Share Information

The most fundamental truth is that we have information on one computer (Computer A) and we want it to be on another computer (Computer B)...

🔑 Principle 2: The Problem of Scale

When more computers join, connecting every machine directly becomes impossible... The solution is a Local Network with a Switch.

🔑 Principle 3: The Problem of Connecting Networks

Networks are connected using a Router, creating the Internet: a "network of networks".

🔑 Principle 4: The Problem of Finding Anyone (Addressing)

Each device needs a unique IP Address, just like every house needs a postal address.

🔑 Principle 5: The Problem of Reliability and Fairness

Large files are split into packets and reassembled using TCP/IP for reliable transfer.

📖 Read Full Explanation (Principles + Summary)

So, the internet isn't magic. It's a series of logical solutions to a series of fundamental problems:

  1. Problem: Share data → Solution: Direct link
  2. Problem: Connect many computers → Solution: Local network with Switch
  3. Problem: Connect many networks → Solution: Routers
  4. Problem: Find specific computers → Solution: IP Address
  5. Problem: Reliable communication → Solution: TCP/IP Packets

🔎 IP Address

IPv4

Classic 32-bit addresses like 172.217.16.142. Limited to ~4.3 billion unique addresses.

🌍 Public vs Private IP

  • Public IP: Unique global address, assigned by ISP.
  • Private IP: Used inside local networks (e.g., 192.168.x.x).

IPv6

128-bit addresses (2128 possibilities). Example: 2001:0db8:85a3::8a2e:0370:7334

🖥️ MAC Address

Unique hardware identifier for network devices (48-bit, shown as 3C:22:FB:A3:B4:C5).

How to Find

  • Windows: ipconfig /all or getmac /v
  • macOS: ifconfig

🔢 Port Numbers

Ports identify specific services (0–65535). Three categories:

  1. Well-Known (0–1023): HTTP(80), HTTPS(443), DNS(53).
  2. Registered (1024–49151): MongoDB(27017), MySQL(3306), React Dev (3000).
  3. Dynamic (49152–65535): Temporary, client-side use.

🌐 DNS (Domain Name System)

DNS translates google.com → 142.250.72.206. Steps:

  1. Check cache (browser → OS → router).
  2. Ask Recursive Resolver (ISP or public like Google’s 8.8.8.8).
  3. Resolver queries Root → TLD → Authoritative Name Server.
  4. Answer returned and cached.
  5. Browser connects to IP using port (e.g., 443 for HTTPS).
